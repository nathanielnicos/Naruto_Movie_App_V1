# Naruto Movie App (V1)

## 📌 Project Overview
This project is a movie catalog application developed as part of my learning journey in Android development. Originally based on a curriculum from **Dicoding**, I took the initiative to enhance the project by migrating the entire user interface from the traditional **XML (View-based)** system to **Jetpack Compose**, Android’s modern toolkit for native UI.

## 🚀 Key Features
* **Movie Catalog:** Displays a curated list of Naruto movies with titles and ratings.
* **Detailed View:** Seamless navigation (Intent) to a dedicated detail page for each movie.
* **Static Data Integration:** Hardcoded data management for images, overviews, release dates, and durations.
* **Modern UI:** Rebuilt entirely using declarative UI principles with Jetpack Compose.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Migrated from XML)
* **Design Pattern:** Component-based UI architecture.

## 💡 Learning Objectives
The main goal of this project was to master the fundamental shifts in Android development, specifically:
1. **Migration Logic:** Understanding the transition from imperative (XML) to declarative (Compose) UI.
2. **State Management:** Managing UI states within a Compose environment.
3. **Jetpack Navigation:** Implementing smooth transitions between screens in a single-activity architecture.
